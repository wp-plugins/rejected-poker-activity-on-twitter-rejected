== Description ==

Displays the latest Twitter activity with the term "poker" in it. Great for poker-enthusiasts.

== Installation ==

1. Upload [poker_activity_on_twitter.php] to your [/wp-content/plugins/] directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Click on 'Widgets' under 'Appearance'.
4. Drag the widget to your sidebar.
